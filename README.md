# Projeto de Navegação com ESP32 e App Híbrido

Este projeto contém o código para um sistema de navegação que envia instruções de um aplicativo para um dispositivo ESP32 via Wi-Fi.

## Estrutura

- **/mobile-app**: Contém o aplicativo híbrido/web desenvolvido com React e TypeScript.
- **/esp32-firmware**: Contém o código para o microcontrolador ESP32, desenvolvido em C++ para a IDE do Arduino.

Consulte o `README.md` dentro de cada pasta para instruções específicas de configuração e execução.
